1. скачать по ссылке питон 3 для винды  https://www.python.org/ftp/python/3.12.0/python-3.12.0-amd64.exe
при установке поставить галочку где что-то про PATH

2. в командной строке выполнить поочереди команды

pip install python-telegram-bot
pip install gspread
pip install oauth2client


3. залогиниться в браузере под своим аккаунтом  в гугле и 

1. Перейдите на веб-сайт Google Cloud Console (https://console.cloud.google.com/) и войдите в свой аккаунт Google.
1.1 создать проект
2. Откройте проект, для которого был создан API-ключ.
3. На левой панели навигации выберите "Учетные данные".
4. На странице "Учетные данные" найдите свой API-ключ, который вы создали ранее, когда получали доступ к API Google Sheets.
5. Щелкните на имени API-ключа, чтобы открыть его настройки.
6. В разделе "Тип учетных данных" выберите "Ключ API" и вы увидите подробные сведения о вашем ключе.
7. После прокрутки страницы до раздела "Ключи для приложений" вы увидите кнопку "Создать ключ" - нажмите на нее. Появится выпадающий список со следующими опциями: "Ключ JSON" и "Ключ в формате P12".
8. Выберите "Ключ JSON" и файл JSON с вашим ключом автоматически загрузится на ваш компьютер.
9. разрешить для приложения использовать гугл таблицы

Теперь у вас есть файл JSON с вашим ключом API Google Sheets. переименовать его в credentials.json и полоэить рядом с файлом кода *.py

4. зарегать телеграм бота в BotFather и вставить его TOKEN в код
